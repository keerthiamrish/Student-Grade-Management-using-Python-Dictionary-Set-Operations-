# Student Grade and Hobbies Management using Python

## Project Description

This project demonstrates the use of Python dictionary and set data structures to manage student grades and hobbies.

The program stores student names and their grades using a dictionary and allows basic operations like adding and updating records.

It also uses set and frozenset to handle unique hobby entries.


## Features

## Dictionary Operations

* Create student grade dictionary
* Add new student records
* Update multiple student grades

## Set Operations

* Accept user input for hobbies
* Store hobbies in a set (unique values)
* Convert set to frozenset (immutable set)


## Technologies Used

* Python
* Dictionary
* Set & Frozenset
* User Input

## Sample Output

* Displays student grades dictionary
* Shows updated records
* Displays hobbies set and frozen set

## Key Learnings

* Working with dictionaries
* Adding and updating values
* Understanding set and frozenset
* Handling user input



✨ This project is useful for beginners to understand Python data structures.
